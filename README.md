# OWP Art Reflection Slideshow

Slideshow web app for OWP Art Reflections.

## Development

```
npx http-server -p 8080
```

Port `8080` is important (which is also the default so `-p 8080` can be omitted) as http://locahost:8080 is enabled for CORS on server.

http://localhost:8080

## Deployment

Push to `main` branch to https://github.com/examind-ai/owp-gpt-slideshow to auto-deploy to Netlify.
